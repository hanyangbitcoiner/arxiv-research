# π²: 长上下文推理数据

**论文：** [arXiv:2604.05114](https://arxiv.org/abs/2604.05114)
**作者：** Quyet V. Do et al. (Virginia Tech)
**日期：** 2026-04-06

---

## 核心问题

长上下文推理能力难以提升，缺乏高质量、结构化的推理训练数据。

---

## 核心贡献

π² 流水线构建高质量推理数据：

```
┌─────────────────────────────────────────────────────────┐
│  Step 1: 表格收集与扩展                                  │
│  - 从维基百科提取表格                                    │
│  - 合成扩展新列                                         │
├─────────────────────────────────────────────────────────┤
│  Step 2: QA 对生成                                      │
│  - 多跳分析推理问题                                      │
│  - SQL + Python 双重执行验证                             │
├─────────────────────────────────────────────────────────┤
│  Step 3: 推理轨迹生成                                  │
│  - 反向翻译生成结构化推理步骤                             │
└─────────────────────────────────────────────────────────┘
```

---

## 伪代码实现

```python
class PiSquared:
    def expand_table(self, table, context):
        expanded = table.copy()
        for i in range(3):
            new_col = self.extract_column_from_links(table, context)
            if self.verify_column(new_col):
                expanded.add_column(new_col)
        return expanded

    def generate_reasoning_trace(self, question, answer, context):
        prompt = f"""
        Given: Q={question}, A={answer}, C={context}
        Generate step-by-step reasoning trace
        """
        return self.llm.generate(prompt)
```
